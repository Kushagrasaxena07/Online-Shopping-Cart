# Online Shopping Cart 🛒

A Java-based Online Shopping Cart application that demonstrates Object-Oriented Programming (OOP) concepts.  
The system allows users to browse products, add items to the cart, remove items, apply coupons, and calculate the total bill dynamically.

---

## 🚀 Features
- Browse available products
- Add items to the shopping cart
- Remove items from the cart
- Update product quantity
- Apply discount coupons
- Automatic total price calculation
- Generate checkout receipt

---

## 🛠️ Technologies Used
- **Java**
- **Object-Oriented Programming (OOP)**
- **Java Swing (GUI)**
- **Java Collections (ArrayList, Map)**
